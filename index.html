Convolutional Neural Networks for Irregular Data Points (Superpixels): Breast Cancer Stage Classification

Master's Project, Advisor: Prof. Raghu Machiraju, Ohio State University
Abstract: Breast cancer image classification task has so far been limited to grade and histopathology based categorization. In this project, we predict stage given only the digital whole slide images. Convolutional neural networks are state-of-art models in image classification. However, in previous work, convolutional networks have been used directly on raw pixel values. The entire task of feature extraction is dependent on the model. In this project, we propose to use convolutional neural networks at an intermediate stage on precomputed localized whole slide image features. We treat features corresponding to irregular-shaped homogeneous compartments of the image as data points. The challenge is to exploit the spatial arrangement of these irregular points to create a a viable input for the convolutional neural network. We train and evaluate the methodology on 2800 tiles obtained from the TCGA website.

Presentation

Recipe Visualizer

It is the dream of many to have bots cook dinner after a long day of work. Cbots work with graphs. This project parses and visualizes a cooking recipe in a graph form. First each instruction is divided into action verb, ingredient and agent/location. A graph is created by connecting the ingredients and agents in each instruction set to the action verb. It is assumed that the result of an instruction is being carried forward onto the next instruction.

Music Signal Separation

Humans have the ability to distinguish different tones in a complex sound structure such as human voices or different musical instruments. This model uses recurrent neural networks and soft-masking for music signal separation of monaural, synchronous violin and flute melodies mixed at 0dB. The model outputs signals such that each output signal is predominant in one instrument. This makes it easier for an average (not musically trained) person to figure out and track the melody played by constituting instruments.
