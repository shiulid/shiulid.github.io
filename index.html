Recipe Visualizer

It is the dream of many to have bots cook dinner after a long day of work. Cbots work with graphs. This project parses and visualizes a cooking recipe in a graph form. First each instruction is divided into action verb, ingredient and agent/location. A graph is created by connecting the ingredients and agents in each instruction set to the action verb. Music Signal Separation

Humans have the ability to distinguish different tones in a complex sound structure such as human voices or different musical instruments. This model uses recurrent neural networks and soft-masking for music signal separation of monaural, synchronous violin and flute melodies mixed at 0dB. The model outputs signals such that each output signal is predominant in one instrument. This makes it easier for an average (not musically trained) person to figure out and track the melody played by constituting instruments.

Part-Of-Speech Tagger

Used structured perceptron and Viterbi algorithm for part-of-speech tagging. Also did cross domain testing with Twitter, penn-tree bank and IRC chat data.

Multi-Camera Multiple Object Tracker

Developed a consolidated utility to track multiple objects in multiple networked cameras in real-time. Features included: handshaking between multiple camera views for consistent object labeling, auto-initialization of the TLD (tracking-learning-detection) tracker using GMM based motion detector.
